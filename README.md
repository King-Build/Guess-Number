# 🎯 Guess Number Game

## 🔎 Overview  
A simple number guessing game where the player tries to guess a random number between **1 and 100**.  
The game provides instant feedback and keeps track of score and high score.

## 🛠 Technologies Used  
- HTML  
- CSS  
- JavaScript
